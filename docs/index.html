Rayleigh Benard Convection Simulator
Rayleigh Benard Convection Simulator Documentation

In this project, Rayleigh-Benard convection is simulated in a thin layer of fluid (2D domain).

To the user: before running this code, modify Makefile, line 2, and change the path to a temporary folder. The user must have writing permission to such folder.

Rayleigh-Benard convection is a thermal gradient driven natural convection, occurring in a planar, horizontal layer of fluid heated from below. During this phenomena the fluid develops regular patterns of convection cells called Benard cells.

This simulation works with Firefox.

Simulation demo
  • The linear algebra algorithms are implemented via Eigen.
  • The graphical interface is implemented via raylib.
  • The network and I/O programming are implemented with Boost.Asio.
  • The numerical solution of the Navier-Stockes equation and the set of default parameter values is tacken from: Suraj Shankar (2022). Rayleigh Benard Convection, MATLAB Central File Exchange. Retrieved May 29, 2022.

The simulation is run according to the following hypotesis:

  • Bottom and top walls are rigid, impermeable and isothermal.
  • Side walls are adiabatic.
  • No-slip is enforced for velocity on all sides.
  • Homogenous boundary conditions are employed for pressure.
  • Boussinesq approximation is used to model the bouyancy driven flow.
